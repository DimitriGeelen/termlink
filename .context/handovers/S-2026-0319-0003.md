## Decisions Made This Session

1. **Split PTY writes per KeyEntry with 10ms delay (T-178)**
   - Why: Claude Code TUI swallows combined writes; discrete calls with delay are required for keystroke delivery.
   - Alternatives rejected: Single write with \r appended (confirmed broken), OS-level uinput (too invasive).

2. **TOFU TLS model for cross-hub trust (T-179/T-182)**
   - Why: No CA infrastructure available in dev environments; TOFU matches user mental model from SSH.
   - Alternatives rejected: Self-signed CA (requires distribution), mTLS (operational overhead), skip verify (insecure).

3. **KnownHubStore persisted to ~/.termlink/known_hubs (T-182)**
   - Why: Per-user trust store mirrors SSH known_hosts placement; survives process restarts.
   - Alternatives rejected: In-memory only (no persistence), project-local store (breaks multi-project use).
