How it works
  • Ollama — Local models. No API key needed. Models must be pulled first (ollama pull model).
  • OpenRouter — Cloud gateway to 200+ models (Claude, GPT, Gemini, Llama, etc). Requires an API key from openrouter.ai/keys.
  • Key storage — Keys are encrypted with Fernet (machine-bound via /etc/machine-id). Environment variables override stored keys.
  • Hot-switching — Changing the provider takes effect immediately, no restart needed.
